# Fake GNOME Keyring
Fake GNOME Keyring - поддельное окно разблокировки связки ключей GNOME, которое выполнит команду из 
~/.config/fake-gnome-keyring/config.json, в случае ввода неправильного пароля/закрытия окна. <br>
